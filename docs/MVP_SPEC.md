## 1. MVP概要

### 1.1 プロダクトビジョン
**「AI入力 + 人間承認」のハイブリッド型フォーム営業支援システム**

企業のお問い合わせフォームへの営業アプローチを半自動化し、効率的な新規顧客開拓を実現します。

### 1.2 ターゲットユーザー
| ユーザー | ニーズ | 提供価値 |
|----------|--------|----------|
| **BtoB営業チーム** | 手動フォーム入力の効率化 | 1件2分 → 30秒（75%削減） |
| **営業代行会社** | 大量案件の処理能力向上 | 月6万件対応可能 |
| **ワーカー** | 単純作業からの解放 | reCAPTCHA対応に集中 |
| **管理者** | 案件・タスク一元管理 | 進捗リアルタイム可視化 |

### 1.3 解決する課題
1. **手動作業の非効率性**: 1件5-10分 → フォーム入力は自動化
2. **reCAPTCHA問題**: 完全自動化不可 → 人間が対応（半自動化）
3. **スケーラビリティ**: 人手依存 → ハイブリッド自動化で処理能力向上
4. **品質の不安定性**: 手動入力ミス → AI生成＋自動入力で品質維持
5. **コスト**: 全て手動対応 → 55%を自動化（提案中）

### 4.2 画面別要素

#### admin-phase2a.html（管理コンソール）
**タブ構成:**
- **企業管理**: 企業情報の登録・編集・削除
  - フィールド: 企業名、URL、業界、規模、お問い合わせURL
- **案件管理**: 商材情報・送信者情報の管理
  - 商材フィールド: 案件名、メッセージテンプレート
  - 送信者フィールド: 名前、メール、会社名、電話番号
- **タスク管理**: タスク生成・進捗確認
  - 統計: 総数、未処理、完了、失敗
  - 一括生成: 企業選択 + 案件選択 → 生成

#### simple-console-v2.html（ワーカーコンソール）
**3カラム構成:**
- **左カラム（タスク一覧）**: pending/in_progress/completedのフィルタ
- **中央カラム（詳細表示）**: 企業情報、案件情報、メッセージ、ボタン（自動入力、送信完了）
- **右カラム（VNCビューア）**: noVNC iframe、ブラウザ画面のリアルタイム表示
