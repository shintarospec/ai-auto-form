## 🎯 サービス概要とAI使用箇所

### DeepBiz（企業情報プラットフォーム）

**役割**: 営業対象企業のWebサイトを解析し、事業内容・強み・課題を構造化データとして提供

| ステップ | 処理内容 | AI使用 | AIモデル | 目的 |
|---------|---------|--------|---------|------|
| Place ID取得 | Google Maps API呼び出し | ❌ 不要 | - | 店舗特定 |
| CID/Website取得 | スクレイピング | ❌ 不要 | - | 基本情報収集 |
| **企業分析** | **Web解析→構造化JSON** | **✅ 使用** | **Gemini 2.0 Flash** | **非構造化→構造化** |
| キャッシュ保存 | DB保存（90日） | ❌ 不要 | - | 再利用 |

**AIコスト**: **0.012円/社**（初回のみ、2回目以降はキャッシュで0円）

---

### TheSide（営業フォーム自動送信）

**役割**: 企業の問い合わせフォームへの自動送信を支援

| ステップ | 処理内容 | AI使用 | AIモデル | 目的 |
|---------|---------|--------|---------|------|
| 企業情報取得 | DeepBiz API呼び出し | ❌ 不要 | - | 外部データ取得 |
| **テンプレート作成** | **商材情報→営業文面** | **✅ 使用** | **GPT-4 Turbo** | **高品質文面生成** |
| **メッセージカスタマイズ** | **企業別パーソナライズ** | **✅ 使用** | **Gemini 2.5 Flash** | **大量処理最適化** |
| **フォームマッピング** | **フォーム構造解析** | **✅ 使用** | **Gemini 2.5 Flash** | **可変フォーム対応** |
| フォーム送信 | Playwright自動化 | ❌ 不要 | - | ブラウザ操作 |

**AIコスト**: 
- テンプレート: **3.4円/案件**
- カスタマイズ: **0.007円/件**
- フォームマッピング: **0.010円/社**（初回のみ）

### TheSide: 営業自動化コスト

#### 1. テンプレート作成（GPT-4 Turbo）

**頻度**: 案件数分のみ（月3〜10件程度）

```
【入力】
商材情報: Shopify移行サポート
詳細: ECサイトのShopify移行を総合支援...（約300文字）
約1,000トークン
  ↓ 🤖 GPT-4 Turbo
【出力】
高品質営業文面テンプレート（約400-500文字）
約1,000トークン
```

**コスト計算**:
```
入力: 1,000 × $0.01 = $0.01
出力: 1,000 × $0.03 = $0.03
────────────────────────────────
合計: $0.04 ≈ 3.4円/案件

月10案件: 34円/月
```

**なぜGPT-4を使うか**:
- テンプレートの品質が全体の成約率を左右
- 1案件で100社以上に送信するため、初期品質が重要
- 月10回程度なら高コストでも許容範囲

【出力】
パーソナライズされた営業文面（約400文字）
約800トークン
```

**コスト計算**:
```
入力: 1,200 × $0.00001875 = $0.000023
出力: 800 × $0.000075 = $0.000060
────────────────────────────────
合計: $0.000083 ≈ 0.007円/件
```

**スケール別コスト**:
| 送信数 | 月額コスト | 1件単価 |

| 1,000件 | 7円 | 0.007円 |

| 100,000件 | 700円 | 0.007円 |

**なぜGemini Flashを使うか**:

## 💡 コスト最適化戦略

### 1. DeepBiz側の最適化

#### キャッシュ戦略

```
【1回目】
100社分析: 100 × 0.012円 = 1.2円

【2回目以降（90日以内）】
同じ100社: 0円（キャッシュヒット）

【複数ユーザー共有】
ユーザーA: 100社分析（1.2円）
ユーザーB: 同じ100社（0円）← キャッシュ共有
ユーザーC: 同じ100社（0円）← キャッシュ共有
────────────────────────────────
実効コスト: 1.2円 ÷ 3ユーザー = 0.4円/ユーザー
```

**効果**: 複数ユーザー運用で**実効コストが1/N**に削減

---

### 2. TheSide側の最適化

#### ハイブリッド戦略

```
【テンプレート作成】
GPT-4 Turbo: 3.4円/案件
頻度: 月3〜10件
→ 品質重視、コストインパクト小

【カスタマイズ】
Gemini Flash: 0.007円/件

→ コスト最優先、品質は十分

All GPT-4: 30,000件 × 3.4円 = 102,000円/月

ハイブリッド: 10.2円 + 210円 = 220円/月

→ ハイブリッドはAll Geminiより+10円で最高品質

**結論**: **月10円の追加投資で、テンプレート品質が劇的向上**

## 📊 ROI分析

### 手動処理との比較

#### コスト比較（月10,000社 × 3案件 = 30,000件）

| 項目 | 手動処理 | AI自動化 | 削減額 |
|------|---------|---------|--------|
| **企業情報調査** | 83.3時間<br>(10,000社 × 5分) | 0時間<br>(DeepBiz API) | 83.3時間 |
| **テンプレート作成** | 1.5時間<br>(3案件 × 30分) | 0.01時間<br>(GPT-4) | 1.5時間 |
| **メッセージカスタマイズ** | 2,500時間<br>(30,000件 × 5分) | 2.1時間<br>(Gemini Flash) | 2,498時間 |

---

#### 金銭的ROI（時給2,000円換算）

```
【手動処理コスト】
2,752時間 × 2,000円 = 5,504,000円/月

【AI自動化コスト】
AI: 440円/月

【削減額】

```

**投資回収期間**: **即時**（初月から黒字）

### 成約率向上による追加価値

```
【前提】
送信数: 30,000件/月
手動時の返信率: 2%
AI時の返信率: 3%（パーソナライズ効果）
商談単価: 100,000円

【効果】
手動: 30,000 × 2% × 100,000円 = 60,000,000円/月

差額: 30,000,000円/月

【AIコスト】
440円/月

【費用対効果】
30,000,000円 ÷ 440円 = 68,182倍
```

**結論**: パーソナライズによる返信率1%向上で、**月3,000万円の追加売上**
